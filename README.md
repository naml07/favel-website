Site web premium pour FAVEL, agence de communication de l'Afrique de l'Ouest francophone.

## Pages
- `index.html` — Accueil
- `produits.html` — Produits
- `packs.html` — Packs
- `ebooks.html` — Ebooks
- `notre-histoire.html` — Notre Histoire
- `contact.html` — Contact

- CSS3 (design system, animations, glassmorphism)
- JavaScript Vanilla ES6+
